# Engineering Industries

Custom FDM 3D Printing & CNC Wood Routing services website.

## Stack

- **React** (`.jsx`) — primary version, component-based
- **Plain HTML** (`index.html`) — standalone fallback, no build step required

## Sections

- Hero with live capability cards
- Scrolling ticker
- Services (FDM 3D Printing + CNC Wood Routing + Design Help)
- 4-step process
- Materials & specifications table
- Pricing tiers
- Gallery
- FAQ accordion
- Quote request form / Contact
- Printable flyer page

## Materials Offered

**FDM 3D Printing:** PLA · PETG · ABS · TPU · ASA  
**CNC Wood:** Baltic Birch Plywood · MDF · Oak · Maple · Walnut · Cedar · Pine

## Deploying

### React (recommended)
Drop `EngineeringIndustries.jsx` into any React project (Vite, CRA, Next.js).  
Uses only `useState` and `useEffect` — no external dependencies.

### Plain HTML
Open `index.html` directly in a browser, or deploy to any static host (Cloudflare Pages, Netlify, GitHub Pages).

## Hosting (free options)

- **Cloudflare Pages** — drag & drop, `yoursite.pages.dev` subdomain
- **Netlify** — drag & drop, `yoursite.netlify.app` subdomain  
- **GitHub Pages** — enable in repo Settings → Pages → deploy from `main`
